2. membuat render untuk navigation dengan nama _application_nav.html.erb di app/views/shared/_application_nav.html.erb

- lalu pada views/layouts/application.html.erb 
kita ubah code render ini :

<%= render "shared/nav", location: 'top' %>

menjadi :

<%= render "shared/application_nav" %>

- buka views/pages/home.html.erb

          <div class="masthead clearfix">

- buka views/shared/_application_nav.html.erb

          <div class="masthead clearfix">
