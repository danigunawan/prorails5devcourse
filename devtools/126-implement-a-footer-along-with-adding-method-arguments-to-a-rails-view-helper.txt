CARA 1 - Navlink helpers
========

dengan cara ini kita bisa langsung memanggil class css dari helpers
jadi kita mendefinisikan isi dari  <%= login_helper %> pada application_helper.rb 
pada action login_helper

- buka app/helpers/application_helper.rb

ubah pada action login_helper dengan menambahkan class nav-link agar css bisa match dan rapih

# dari :

def login_helper
		if current_user.is_a?(GuestUser)
			(link_to "Register", new_user_registration_path) + "<br>".html_safe +
			(link_to "Login", new_user_session_path)
		else
			link_to "Logout", destroy_user_session_path, method: :delete
		end
	end

# menjadi :


	def login_helper
		if current_user.is_a?(GuestUser)
			(link_to "Register", new_user_registration_path, class: 'nav-link') + " ".html_safe +
			(link_to "Login", new_user_session_path, class: 'nav-link')
		else
			link_to "Logout", destroy_user_session_path, method: :delete, class: 'nav-link'
		end
	end


CARA 2 - Navlink helpers
========
dengan cara ini kita bisa langsung memanggil class css langsung dari local variables pass 
yg asalnya dari <%= login_helper %> kita pass menjadi <%= login_helper 'nav-link'%> 


- buka app/helpers/application_helper.rb

# ubah dari :

def login_helper
		if current_user.is_a?(GuestUser)
			(link_to "Register", new_user_registration_path) + "<br>".html_safe +
			(link_to "Login", new_user_session_path)
		else
			link_to "Logout", destroy_user_session_path, method: :delete
		end
	end

# menjadi :

	def login_helper style
		if current_user.is_a?(GuestUser)
			(link_to "Register", new_user_registration_path, class: style) + " ".html_safe +
			(link_to "Login", new_user_session_path, class: style)
		else
			link_to "Logout", destroy_user_session_path, method: :delete, class: style
		end
	end


### PADA action def login_helper style (style sendiri adalah sebuah identitas/pengenal/penghubung untuk parameter variabel passing yang dipakai)
 
- buka _application_nav.html.erb
# ubah dari :
 <%= login_helper %>

# menjadi :

 <%= login_helper 'nav-link' %>
